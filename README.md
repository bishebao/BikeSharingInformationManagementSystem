<h2>一、项目简介</h2>
基于SpringBoot Vue共享单车信息管理系统

角色：管理员、用户

<h2>二、项目技术</h2>
前端 jsp、jquery、bootstrap，后端 maven、springmvc、spring、mybatis
<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">

<strong>用户：</strong>
系统首页、个人中心、单车租赁管理、租赁归还管理、报修信息管理、检修信息管理等功能进行相应操作

<strong>管理员：</strong>
